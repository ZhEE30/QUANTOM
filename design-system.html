Design System

Building a seamless, efficient, and appetizing management experience for the fast food industry.

Visual Principles

A minimalist design that stays out of the way, allowing high-quality visuals and data to stand out.

Clarity

Text is legible and icons are precise to ensure speed during high-traffic kitchen hours.

Depth

Visual layers and soft shadows differentiate background tasks from active ordering chits.

Consistency

Uniform button radii and standard orange accents create a predictable user journey.

Color

Our palette uses "Appetite Colors" designed to evoke freshness while maintaining professional legibility.

Muted Orange #FF9100

Action items, CTA buttons, and brand emphasis.

Deep Espresso #2D1B14

Headings, navigation text, and primary containers.

Broken White #F8F9FA

Section backgrounds and secondary layout containers.

Typography

Montserrat provides a geometric and modern feel. It is utilized in various weights to establish a clear information hierarchy.

Layout & Grid

The system utilizes a 12-column grid. Key spacing rules include:

  • Safe Areas: 15px-20px padding for mobile touch targets.
  • Section Vertical Padding: 80px-100px to maintain breathing room.
  • Max Width: Content containers are capped at 1200px for desktop readability.

Iconography

Following the HIG principle of Simplicity, icons in QUANTOM are used to provide immediate visual context without needing to read text. We utilize the Bootstrap Icons set for consistency and scalability.

Usage Rules
  • Scale: Icons should generally be sized at 24px for standard UI and 32px-40px for feature highlights.
  • Color: Icons should inherit the text color or use #FF9100 (Primary Orange) to denote interactivity or key features.
  • Weight: Use "stroke" versions rather than "filled" to maintain a light, modern aesthetic.
